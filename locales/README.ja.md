# <a href="https://www.youtube.com/watch?v=PnK4gzO6S3Q" target="_blank"><img src="https://user-images.githubusercontent.com/39828164/194585524-d49a7908-044f-4f02-8c9b-0b6f557d4aac.png" width="100%" alt="Welcome to Hack Club!"></a>

高校ハッカーの世界規模のコミュニティへようこそ。私たちはアーティスト、ライター、エンジニア、DIY愛好家、キャンパー、映画監督、ボランティアです。私たちは**物事を作ります**。私たちは**お互いを助けます**。私たちは**楽しんでいます**。**[私たちに参加してください ➚](https://hackclub.com/slack)**

このリポジトリは、私たちの[ワークショップ](https://workshops.hackclub.com)を保管する場所です。ワークショップは、自己主導型のコーディング学習チュートリアルであり、また私たちの[行動規範](https://hackclub.com/conduct/)も含まれています。

### 🌈 参加する

コミュニティに参加していただけると嬉しいです：

- [Hack Clubを開始](https://hackclub.com) して、あなたの高校で。

- [Hack Club Slack](https://hackclub.com/slack) に参加し、数千人の若手クリエイターとチャットしましょう。

- あなたの地域で_(または開始!)_ [高校ハッカソン](https://hackclub.com/hackathons) に参加しましょう。

- [🏦 HCB](https://hackclub.com/hcb) を使って、あなたのプロジェクトを501(c)(3)非営利団体にしましょう。

- [私たちのワークショップ](https://workshops.hackclub.com) のいずれかを使って、新しいプロジェクトを完成させましょう。

### [🦦](https://shakingmy.horse/) GitHubでの貢献

GitHubでは、多くの方法で貢献できます：

- [クラブやハッカーが構築できるHack Clubワークショップ](https://workshops.hackclub.com/submit-a-workshop/)を書く。

- [🦖恐竜を描画](https://hack.af/draw-dino)し、[`hackclub/dinosaurs`](https://github.com/hackclub/dinosaurs)へのプルリクエストを提出する。

- [🍃 Sprig](https://sprig.hackclub.com)でゲームを作成する（[コンソール](https://github.com/hackclub/sprig/tree/main/hardware)をあなたに送付します！）。

- 多言語対応？[サイトの翻訳](https://github.com/hackclub/global/issues/15)にご協力ください！

[`contribute.hackclub.com`](https://contribute.hackclub.com)で、よりアクティブなリポジトリや貢献ガイドをご覧ください。

## ✨ 我々の哲学

私たちは、コードを学ぶことは、スーパーパワーを獲得することに独特であると考えています。それは、あなたを_消費者_から_創造者_へと変えるもので、あなたのコンピュータを創造のための道具に変えるものです。もしもあなたがこれを読んでいるなら、アプリを構築する方法を学ぶことができます。創造するための最適なタイミングは今です。

**Hack Clubの目標は、あなたのスーパーパワーを発展させることです。** インターネット上とすべての学校で、人々が毎週コードを使って興味深いものを作る、包摂的な場所を提供したいと考えています。[オンラインSlack](https://hackclub.com/slack)（20,000人以上の学生が所属するDiscordスタイルのオンラインチャットルーム）では、コーディングに関する質問をしたり、素晴らしい友人を出会ったり、[構築しているプロジェクト](https://scrapbook.hackclub.com)を共有したり、その他にも多くのことができます。

Hack Clubは、ティーンエイジャーのハッカーによって設立され、ティーンエイジャーのために存在するコミュニティです。我々のソフトウェア、過去のイベント、ブランドなどは、世界中のティーンエイジャーによって作られています。また、我々は完全に[オープンソース](https://hackclub.com/opensource/)の組織です。[我々のウェブサイト](https://github.com/hackclub/site)から[我々の財務](https://bank.hackclub.com/hq)に至るまで、すべてが公開されています。
