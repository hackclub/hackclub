# <a href="https://www.youtube.com/watch?v=PnK4gzO6S3Q" target="_blank"><img src="https://user-images.githubusercontent.com/39828164/194585524-d49a7908-044f-4f02-8c9b-0b6f557d4aac.png" width="100%" alt="Welcome to Hack Club!"></a>

Willkommen in einer weltweiten Gemeinschaft von Highschool-Hackern. Wir sind Künstler, Schriftsteller, Ingenieure, Bastler, Camper, Filmemacher, Freiwillige. Wir **machen** Dinge. Wir **helfen** einander. Wir haben **Spaß**. **[Schließe dich uns an ➚](https://hackclub.com/slack)**

Dieses Repository ist der Ort, an dem wir unsere [Workshops](https://workshops.hackclub.com), die selbstgeführten Lern-Code-Tutorials, sowie unsere [Code of Conduct](https://hackclub.com/conduct/) speichern.

### 🌈 Mitmachen

Wir würden uns sehr freuen, wenn du in der Community mitmachen würdest:

- [Starte einen Hack Club](https://hackclub.com) an deiner High School.

- Melde dich bei dem [Hack Club Slack](https://hackclub.com/slack) an und chatte mit tausenden anderen jungen Makers.

- Besuche _(oder starte!)_ einen [High School Hackathon](https://hackclub.com/hackathons) in deiner Region.

- Mache dein Projekt zu einer gemeinnützigen Organisation (501(c)(3)) mit [🏦 HCB](https://hackclub.com/hcb).

- Bringe ein neues Projekt mit einem unserer [Workshops](https://workshops.hackclub.com) zum Leben.

### [🦦](https://shakingmy.horse/) Beim Mitmachen auf GitHub

Hier auf GitHub gibt es viele Wege, mitzumachen:

- [Schreibe einen Hack Club Workshop](https://workshops.hackclub.com/submit-a-workshop/) für Clubs & Hacker, um daraus zu bauen.

- [Zeichne ein 🦕 Dino](https://hack.af/draw-dino) und sende eine PR an [`hackclub/dinosaurs`](https://github.com/hackclub/dinosaurs).

- Erstelle ein Spiel in [🍃 Sprig](https://sprig.hackclub.com) (wir schicken dir eine [Konsole](https://github.com/hackclub/sprig/tree/main/hardware)!).

- Mehrsprachig? Hilf uns, [unsere Seite zu übersetzen](https://github.com/hackclub/global/issues/15)!

Schau dir [`contribute.hackclub.com`](https://contribute.hackclub.com) an, um mehr aktive Repositories und einen Leitfaden zum Mitmachen zu finden.

## ✨ Unsere Philosophie

Wir denken, das Lernen von Code ist einzigartig wie das Erwerben einer Superkraft: Es verwandelt dich von einem _Verbraucher_ in einen _Ersteller_, indem es deinen Computer in ein Werkzeug für die Schaffung verwandelt. Wenn du dies liest, kannst du lernen, eine App zu erstellen—es ist noch nie einen besseren Zeitpunkt für das Erstellen gegeben haben.

**Das Ziel von Hack Club ist es, dir dabei zu helfen, deine Superkräfte zu entwickeln.** Wir möchten einen inklusiven Raum im Internet und an jeder Schule, an dem Menschen jede Woche interessante Dinge mit Code erstellen. In unserem [online Slack](https://hackclub.com/slack) (Discord-artiges Online-Chatzimmer mit über 20K Studenten-Mitgliedern) kannst du Fragen zu Code stellen, unglaubliche Freunde kennenlernen, [Projekte, an denen du arbeitest](https://scrapbook.hackclub.com) teilen und viel mehr.

Hack Club ist eine Gemeinschaft für und von Teenager-Hacker. Unsere Software, vergangene Veranstaltungen, Markenidentität usw. wurden alle von Teenagern aus der ganzen Welt geschaffen. Wir sind auch eine vollständig [Open-Source](https://hackclub.com/opensource/)-Organisation—[unsere Website](https://github.com/hackclub/site) bis hin zu [unseren Finanzen](https://bank.hackclub.com/hq) sind öffentlich.
