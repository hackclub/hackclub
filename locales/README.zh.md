# <a href="https://www.youtube.com/watch?v=PnK4gzO6S3Q" target="_blank"><img src="https://user-images.githubusercontent.com/39828164/194585524-d49a7908-044f-4f02-8c9b-0b6f557d4aac.png" width="100%" alt="Welcome to Hack Club!"></a>

欢迎加入全球高中生黑客社区。我们是艺术家、作家、工程师、爱好者、露营者、电影制作人、志愿者。我们**创造**事物。我们**互相帮助**。我们**玩得开心**。**[加入我们 ➚](https://hackclub.com/slack)**

此仓库用于存储我们的[Workshops](https://workshops.hackclub.com)，这些是自助式学习编程教程，以及我们的[行为准则](https://hackclub.com/conduct/)。

### 🌈 参与进来

我们非常希望你能参与社区：

- 在你的高中[启动一个Hack Club](https://hackclub.com)。

- 加入[Hack Club Slack](https://hackclub.com/slack)并与数千名其他年轻创造者聊天。

- 参加（或启动！）你所在地区的[高中黑客马拉松](https://hackclub.com/hackathons)。

- 通过[🏦 HCB](https://hackclub.com/hcb)将你的项目变为501(c)(3)非营利组织。

- 通过[我们的工作坊](https://workshops.hackclub.com)发布一个新项目。

### [🦦](https://shakingmy.horse/) 在 GitHub 上贡献

在 GitHub 上，有多种方式可以贡献：

- 为俱乐部和黑客编写 [Hack Club 工作坊](https://workshops.hackclub.com/submit-a-workshop/)。

- 绘制一只 🦕 恐龙并提交一个 PR 到 [`hackclub/dinosaurs`](https://github.com/hackclub/dinosaurs)。

- 在 [🍃 Sprig](https://sprig.hackclub.com) 中创建一个游戏（我们会寄送你一个 [控制台](https://github.com/hackclub/sprig/tree/main/hardware)!）。

- 会多种语言？帮助我们 [翻译我们的网站](https://github.com/hackclub/global/issues/15)！

查看 [`contribute.hackclub.com`](https://contribute.hackclub.com) 以获取更多活跃的仓库和贡献指南。

## ✨ 我们的理念

我们认为学习编程就像获得一种超能力一样独特：它将你从一个_消费者_转变为一个_创造者_，将你的电脑变成创造的工具。如果你正在阅读这篇文章，你就可以学习如何构建一个应用程序——现在是历史上最好的时机。

**Hack Club的目标是帮助你开发你的超能力。** 我们希望在互联网上和每所学校中，创造一个包容的空间，让每个人每周都能用代码制作有趣的东西。在我们的[在线Slack](https://hackclub.com/slack)（一个拥有20,000+学生成员的类似Discord的在线聊天室），你可以提问编程问题，结识令人惊叹的朋友，分享你正在构建的[项目](https://scrapbook.hackclub.com)，还有更多。

Hack Club是一个由青少年黑客建立并为青少年黑客服务的社区。我们的软件、过去的活动、品牌等，都是由来自世界各地的青少年精心打造的。我们也是一个完全[开源](https://hackclub.com/opensource/)的组织——从[我们的网站](https://github.com/hackclub/site)到[我们的财务](https://bank.hackclub.com/hq)都是公开的。
